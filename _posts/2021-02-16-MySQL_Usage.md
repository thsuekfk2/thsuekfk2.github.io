## MySQL Usage



```c
mysql> show databases; 

생성된 이름을 보여준다
```





```
mysql> create schema company; 

 company라는 DB 를 생성한다 
```





```
mysql> create database company;

 company라는  DB 를 생성한다
```







```
mysql> show databases; 

 company DB가 새로 생성되었는지 확인한다
```







```
mysql> use company; 

생성된 여러 중에 사용할 DB를 선택한다
```







```
mysql> show tables; 

DB 내에 table들을 보여준다
```







```
mysql> create table .... ; 

table 을 생성한다
```







```
mysql> show tables; 

table이 새로 생성되었는지 확인한다
```







```
mysql> describe employee; 

 employee table 의 정보를 보여준다
```
