#### **`vue cli 설치`**

npm을 사용하여 Vue CLI를 설치하는 명령어

```
npm install -g @vue/cli
```







Vue 프로젝트를 생성

```
vue create example
```





지정한 폴더 이동

```
cd example
```





서버를 실행

```
npm run serve
```
