#### **`cmd로 visual studio code 열기`**

#### **`vue cli 설치`**

npm을 사용하여 Vue CLI를 설치하는 명령어

```
npm install -g @vue/cli
```



Vue 프로젝트를 생성

```
vue create example
```

![image-20210201034450546](C:\Users\User\AppData\Roaming\Typora\typora-user-images\image-20210201034450546.png)



지정한 폴더 이동

```
cd example
```



서버를 실행

```
npm run serve
```
